# Citation and License Footer

Every translation ends with a citation block and a license notice, preceded by a `<hr class="section-divider">`.

## Scientific Spaces Format

```markdown
<hr class="section-divider">

*Citation: Su, J. (YYYY, Month Day). 中文标题 [English Title]. Scientific Spaces. [https://kexue.fm/archives/XXXXX](https://kexue.fm/archives/XXXXX)*

*Original content licensed under [CC BY-NC-SA 4.0](https://creativecommons.org/licenses/by-nc-sa/4.0/). This translation is shared under the same license.*
```

## Zhihu Format

```markdown
<hr class="section-divider">

*Citation: Author Last, F. (YYYY, Month Day). 中文标题 [English Title]. Zhihu. [https://zhuanlan.zhihu.com/p/XXXXX](https://zhuanlan.zhihu.com/p/XXXXX)*

*Original content shared on Zhihu. This translation is provided for educational purposes.*
```

## Rules

- The citation uses an APA-like format with the original Chinese title followed by the English translation in square brackets.
- The URL appears both as display text and as the link target.
- Scientific Spaces content is licensed CC BY-NC-SA 4.0 — always include this.
- For Zhihu, use the generic educational-purposes notice unless a specific license is stated.
- Both lines are in italics (`*...*`).
- There is no trailing newline after the last line.
